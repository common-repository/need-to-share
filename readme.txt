== Description ==

Need To Share is a plugin to show sharing buttons (Facebook, Twitter and Google Plus) on posts pages as a fixed layer.
You can choose to show sharing buttons always on posts, or to show them only on next posts pages.
You can also choose the text to show above the sharing buttons.


== Installation ==


1. Upload `need-to-share` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin through Need To Share settings

== Frequently Asked Questions ==

= Where can I find the Facebook App ID? =

Go to https://developers.facebook.com/apps/ and create a new App. The App ID is in your new App dashboard.
